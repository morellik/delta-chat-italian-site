Delta Chat è un messaggero sicuro che permette alle persone di chattare tra di loro anche utilizzando server diversi. Questo è solo uno di quelli su cui ti puoi iscrivere ed ha le seguenti caratteristiche:

  • Nessun numero di telefono, indirizzo email, nome reale, ecc. è richiesto per il profilo;
  • 100M di spazio a disposizione;
  • Il server conserva i messaggi per massimo 20 giorni;
  • Ogni utente può inviare fino a 60 email al minuto;
  • La dimensione massima dei messaggi è di 30M;
  • La lunghezza del nome utente (la parte dell'indirizzo prima della @) deve essere di 9 caratteri;
  • La lunghezza della password deve essere di almeno 9 caratteri;
  • Non è consentito l'invio di messaggi in chiaro (non cifrati);
  • Non conserva gli indirizzi IP;
  • Gli indirizzi non utilizzati per 50 giorni vengono distrutti.

Riservatezza

Nessun dato personale richiesto o raccolto


Questo server Chatmail non richiede né conserva informazioni personali. I server Chatmail esistono per trasmettere (archiviare e consegnare) in modo affidabile messaggi cifrati end-to-end tra i dispositivi dell’utente che eseguono l’app di messaggistica Delta Chat. Tecnicamente, potresti pensare a un server Chatmail come un “router di messaggistica” cifrato end-to-end su scala Internet.

Un server Chatmail è molto diverso dai classici server di posta elettronica (ad esempio i server di posta di Google) che richiedono dati personali e memorizzano permanentemente i messaggi. Un server Chatmail si comporta più come il server di messaggistica Signal ma non conosce i numeri di telefono e interopera in modo sicuro e automatico con altri Chatmail e server di posta elettronica classici.

A differenza dei classici server di posta elettronica, questo server di chatmail

  • rimuove incondizionatamente i messaggi dopo 20 giorni,

  • vieta l'invio di messaggi non crittografati,

  • non memorizza indirizzi Internet ("indirizzi IP"),

  • non elabora indirizzi IP in relazione agli indirizzi e-mail.
